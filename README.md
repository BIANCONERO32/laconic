## 🧭 What laconic does

laconic is a token-compression skill for Claude and other AI tools. It helps turn long text into short, clear text that keeps the main point. It uses common words, keeps context in view, and says just enough.

Use it when you want to:

- cut down prompt length
- keep meaning intact
- make notes easier to scan
- shorten text for AI tools
- reduce waste in repeated context
